# Simple Interest Calculator

## Project Description
This project is a simple interest calculator built using a Bash script.
It takes principal, rate of interest, and time period as inputs and calculates the simple interest.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## How to Use
1. Clone the repository
2. Run the script: `bash simple-interest.sh`
3. Enter the required values when prompted
